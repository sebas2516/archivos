Temperatura Baja

La temperatura de un camión es considerada baja, siempre y cuando esta sea menor a los 2℃.

Temperatura Normal

La temperatura de un camión se considera normal, cuando esta se encuentre entre 2℃  y  8℃.

Temperatura Alta

La temperatura de un camión es considerada alta siempre y cuando esta sea mayor a los 8℃.
